# DevOps Learning Path

This repository contains my weekly exercises and projects while learning DevOps, Linux, Networking, Bash, and Cloud technologies.  
It serves as my portfolio to showcase practical skills.

## Structure

- `week1_network/` – Networking basics, ping, traceroute, nslookup
- `week2_linux_basics/` – Linux commands, permissions, file management
- `week3_docker/` – Docker basics and container management
- `resources/` – Useful tutorials and cheat sheets
